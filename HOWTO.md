# QuantumTrade AI — HOWTO: Автономный режим + Safe Mode
> Краткое руководство по GSD-адаптированному workflow

## 🚀 Быстрый старт

### Режимы работы

| Команда | Режим | Описание |
|---------|-------|----------|
| `/auto` | Полный автопилот | Волновое выполнение задач из ROADMAP.md |
| `/wave` | Пошаговый | Одна задача → пауза → подтверждение |
| `/discuss` | Обсуждение | Стратегия без изменения кода |
| `/forensics` | Диагностика | Полный аудит при проблемах |

### Как запустить автономный режим

1. **Открыть Claude Code** в корне проекта
2. **Сказать**: "Запусти /auto" или "Работай по ROADMAP"
3. Claude прочитает STATE.md → определит текущую волну → начнёт работу
4. Каждая волна заканчивается обновлением STATE.md
5. Результат: коммиты в git + обновлённые STATE.md, ROADMAP.md

### Как работает Safe Mode

**Safe Mode = защита от нежелательных изменений при автономной работе**

Уровни защиты:
```
Level 1: Чтение (автоматически)
  → Claude читает код, анализирует, планирует
  → НЕ требует подтверждения

Level 2: Запись (автоматически с проверкой)
  → Claude вносит изменения в код
  → Автоматический syntax check после каждого изменения
  → Запись прогресса в STATE.md

Level 3: Коммит (автоматически)
  → Атомарный коммит с описанием
  → Security scan перед коммитом
  → Откат возможен через git revert

Level 4: Деплой (ТОЛЬКО с разрешения)
  → Claude НИКОГДА не деплоит автоматически
  → Показывает что изменилось, ждёт "деплой" / "go"
  → Pre-deploy: security-auditor agent
```

### Crash Recovery (Восстановление после сбоя)

Если сессия оборвалась:
1. Claude читает STATE.md → видит `in_progress` задачи
2. Проверяет git status → есть ли незакоммиченные изменения
3. Варианты:
   - Продолжить с места остановки
   - Откатить незавершённые изменения
   - Запросить подтверждение пользователя

STATE.md всегда содержит:
- Текущую волну и задачу
- Что было сделано
- Что осталось
- Блокеры и вопросы

### Как оставить бота работать на ночь

1. **Убедись** что последний деплой рабочий: `/diag` в Telegram
2. **Скажи Claude**: "Работай по ROADMAP, волны 1-2, не деплой без меня"
3. **Claude будет**:
   - Писать код по задачам из ROADMAP.md
   - Коммитить каждую завершённую задачу
   - Обновлять STATE.md после каждой волны
   - Останавливаться перед деплоем
4. **Утром**: проверь git log, STATE.md, скажи "деплой" если всё ок
