KuCoin Lending — «Дай монетки взаймы»
Ты даёшь $64 другим трейдерам в аренду → они берут на 7 дней чтобы торговать → каждый день тебе капает % → через 7 дней деньги + проценты возвращаются автоматически

10–50% годовых. Деньги всегда твои, риск почти нулевой.
🪙
ByBit DCI — «Купи дёшево / Продай дорого»
Ставишь $20 на неделю → выбираешь монету (BTC/ETH/SOL) и «цену исполнения» → если цена упала до твоей отметки — автоматически покупаешь монету дёшево → если не упала — забираешь $20 + очень высокий % просто за ожидание

100–900% годовых. Работает неделю, потом заново.
⚡
Funding Rate Arb — «Получай плату за равновесие»
На бирже есть «ставка финансирования» — каждые 8 часов трейдеры лонгов платят трейдерам шортов → ты одновременно покупаешь монету (спот) и открываешь шорт (фьючерс) → цена тебе не важна, ты нейтрален → каждые 8 часов тебе платят ставку автоматически

10–60% годовых. Без риска цены — только доход с ставки.
🎡
ByBit Snowball — «Заработай пока рынок скучает»
Ставишь $20 на диапазон цены → пока цена «прыгает» внутри коридора — ты зарабатываешь % → если цена вышла за коридор («нокаут») — сделка закрывается с прибылью → если цена пробила дно («нокин») — получаешь монету вместо USDT

15–50% годовых. Только в боковом рынке! Сейчас F&G=11 — не подходит.
🌀
Gate.io Lending — «Такой же банк, другая биржа»
То же самое что KuCoin Lending, только на Gate.io → даёшь USDT взаймы маржинальным трейдерам → получаешь % каждый день

8–30% годовых. Код написан ✅, API ключей пока нет ❌.
🗂️
Субаккаунты — «Разные кошельки в одном кармане»
На одной бирже создаёшь несколько «дочерних» аккаунтов → у каждого свои API ключи → бот управляет всеми одновременно → торговля на одном, лендинг на другом, арбитраж на третьем

Польза: изоляция рисков + больше лимитов + параллельные стратегии.
