# /wave — Пошаговое выполнение (Step-Through Mode)
> Как GSD `/gsd next` — выполняет одну задачу, ставит на паузу, ждёт подтверждения

## Протокол
1. Загрузить STATE.md → определить следующую задачу
2. Показать план действий пользователю
3. Ждать подтверждения ("да", "go", "ok")
4. Выполнить ОДНУ задачу
5. Показать результат + что дальше
6. Обновить STATE.md

## Формат вывода
```
🌊 Wave Step #N
📋 Задача: [описание]
📁 Файлы: [список файлов]
⏱ Оценка: ~X мин
🔄 Зависимости: [нет / задача Y]

Выполнить? [да/нет/пропустить]
```

## Safe Mode
- Каждый шаг требует подтверждения
- Показать diff перед коммитом
- Возможность отката: "отмени последний шаг"
