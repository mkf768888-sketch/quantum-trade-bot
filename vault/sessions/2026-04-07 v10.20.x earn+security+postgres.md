- **Важно для будущего:** чтобы активировать webhook secret → вызвать POST /api/setup-webhook

## Ключевые находки

### TG_WEBHOOK_SECRET — вечная ловушка
Эта проблема уже была в v10.0. Шаблон:
1. Кто-то включает TG_WEBHOOK_SECRET enforcement
2. Webhook не перерегистрирован → Telegram не шлёт header
3. Все команды дропаются, исходящие работают
4. Кажется что "бот сломался"
Решение навсегда: webhook secret = warn-only. Security через ADMIN_CHAT_IDS.

### git push забывать нельзя
3 коммита висели локально (v10.20.0, v10.20.1, v10.20.2) → Railway видел старый код → бот на 10.19.9
Правило: после `git commit` всегда сразу `git push origin main`

### DATABASE_URL → Reference Variable
Было: ручная строка подключения в Variables (работает, но нет стрелки на канвасе)
Стало: `${{Postgres.DATABASE_URL}}` → Railway рисует стрелку + автообновляет при rotation

### DCI CRV BuyLow — первое подтверждение
- Позиция: CRV BuyLow $20, APY 507%
- Результат: 94.579464 CRV зачислены на FUND аккаунт ByBit
- При цене ~$0.27/CRV: ~$25.5 (вложено $20 → получено больше, потому что купили на падении)
