# Сессия 2026-04-07 (вечер) — DCI fix + Rebalancer + стратегия роста

## Что задеплоено

### v10.20.4 — DCI "Invalid select price" fix
**Баг:** `dci_auto_place_idle` после первой же ошибки "Invalid select price" делал `return` — останавливал всё. 140 продуктов найдено, LDO @ 1108% APY — ошибка → выход. Остальные 139 продуктов не пробовались.

**Фикс 1:** Добавлены `"invalid select price"`, `"select price"` в `_skip_errors` → теперь `continue` к следующему продукту вместо `return`.

**Фикс 2:** Лимит попыток: 5 → 10 (топ-5 могли все иметь устаревший selectPrice).

**Коммит:** `4e57d23`

### v10.20.5 — KC Flex→Lending авторебалансировщик
**Проблема:** $107 в KC Flex Earn @ 0.7% APY. KC Lending @ 10-50% APR. Бот никогда не перекладывал.

**Новая функция:** `kc_flex_to_lending_rebalance()`
- Держит `KC_FLEX_EARN_BUFFER_USDT=$15` в Flex (для мгновенных BUY ордеров)
- Погашает излишек (~$92) из Flex Earn
- Авто-размещает в KC Lending
- Только если Lending пуст (не дублирует)
- Гистерезис $10 (не ребалансирует по мелочи)
- Запускается каждый час (earn_monitor_loop каждые 4 цикла)
- Присылает Telegram уведомление с diff

**Новая команда:** `/rebalance` — принудительный запуск + отчёт по текущему распределению

**`LENDING_MAX_USDT` default:** 80 → 100

**Ожидаемый эффект:**
- KC Flex: $107 → $15 (буфер)
- KC Lending: $0 → $92 @ 10-50% APR
- Доход: $0.06/мес → ~$2/мес только на этом

**Коммит:** `6a73bca`
