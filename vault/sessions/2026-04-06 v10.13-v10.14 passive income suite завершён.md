# 2026-04-06 — v10.13–v10.14: Passive Income Suite завершён

## Контекст сессии
Продолжение работы после предыдущей сессии (v10.12.6 — Funding Arb hardening).
Цель: реализовать три ключевых продукта пассивного дохода, которые обсуждались как Wave 1.

## ✅ Что сделано

### v10.12.7 — Funding Arb hardening (завершение из прошлой сессии)
- Qty step precision: `_BYBIT_PERP_QTY_STEPS` + `_round_perp_qty()` — фиксит invalid lot sizes
- Rollback продаёт точное кол-во spot (до этого мог продать ВСЕ монеты на счёте)
- Позиции сохраняются в PostgreSQL (`funding_arb_positions` table) — переживают Railway restart
- `_funding_arb_update_collected()` — отслеживает реальный funding через ByBit SETTLEMENT

### v10.13.0 — KuCoin Lending Pro (ЗАВЕРШЁН)
- `kucoin_lending_get_market_rate()` — GET `/api/v1/margin/market` — текущие ставки
- `kucoin_lending_place_order()` — POST `/api/v1/margin/lend` — размещение
- `kucoin_lending_get_active_orders()` — активные займы
- `kucoin_lending_cancel_order()` — отмена
- `kucoin_lending_auto_place()` — авто-логика: rate > LENDING_MIN_APR → размещаем по market-1%
- `/lending` Telegram команда → `_tg_lending()`
- Подключён в `earn_monitor_loop` каждый 4й цикл (~60 мин)
- Railway vars: `LENDING_ENABLED`, `LENDING_MIN_APR=10.0`, `LENDING_MAX_USDT=30.0`, `LENDING_TERM_DAYS=7`

### v10.14.0 — ByBit Snowball (НОВЫЙ)
- Structured product: range-bound, knock-in/knock-out механика
- Principal-protected (USDT всегда возвращается)
- API паттерн: `/v5/earn/advance/product` с `category=Snowball` (аналог DCI/DoubleWin)
- `bybit_snowball_get_products()` — список продуктов с APY + knock-in/knock-out ценами
- `bybit_snowball_get_positions()` — активные позиции
- `snowball_auto_place()` — **умная логика**: F&G 25-70 → sideways рынок → размещаем; за пределами → пропускаем
- `/snowball` Telegram команда
- Railway vars: `SNOWBALL_ENABLED`, `SNOWBALL_MIN_APY=15.0`, `SNOWBALL_MAX_USDT=20.0`

### v10.14.0 — Yield Router v2 (НОВЫЙ)
- Полное APY-сравнение 7 продуктов в одном вызове:
  1. ByBit DCI (100-900% APY)
  2. Funding Rate Arb (переменная)
  3. ByBit Snowball (10-50%)
  4. ByBit DoubleWin (5-30%)
  5. KuCoin Lending (5-50%)
  6. ByBit Flex Earn (0.5-5%)
  7. KuCoin Flex Earn (0.5-3%)
- Режим-зависимые коэффициенты:
  - Snowball: ×1.3 в боковике, ×0.4 в тренде
  - DCI: ×1.2 в тренде
  - Funding Arb: ×1.1 в боковике
- `yield_router_v2_scan()` → возвращает ranking + recommendation
- `/yrouter` Telegram команда
- Логируется каждый час в earn_monitor_loop

## Архитектурный паттерн Passive Income Suite
```
earn_monitor_loop (каждые 15 мин)
  ├── [каждый цикл]  dci_check_settlements() + auto-reinvest
  ├── [каждый цикл]  ByBit FUND → UNIFIED auto-transfer
  └── [каждый 4й (~1ч)]
        ├── dci_auto_place_idle()          ← DCI (приоритет #1, 100-900% APY)
        ├── double_win_auto_place()        ← DoubleWin (если DOUBLE_WIN_ENABLED)
        ├── funding_arb_auto_check()       ← Funding Arb (если FUNDING_ARB_ENABLED)
        ├── kucoin_lending_auto_place()    ← KuCoin Lending (если LENDING_ENABLED)
        ├── snowball_auto_place()          ← Snowball (если SNOWBALL_ENABLED)
        ├── yield_router_v2_scan()         ← логирует лучшую возможность
        └── send_hourly_report()           ← Telegram авто-отчёт
```
