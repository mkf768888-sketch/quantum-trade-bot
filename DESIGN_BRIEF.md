# QuantumTrade AI — Design Brief v1.0
> Для: Claude Code (Квантовый Девелопер)
> Задача: Разработать 5-8 визуальных концептов для Mini App (index.html)
> Дата: 2026-04-05

---

## Контекст проекта

**QuantumTrade AI** — торговый бот с Telegram Mini App. Это основной интерфейс пользователя.

**Технические ограничения:**
- Telegram WebView (iOS + Android)
- Один HTML-файл (всё inline: CSS + JS)
- Тёмная тема обязательна (Telegram тёмная тема)
- Нет внешних CDN шрифтов (только system fonts или Google Fonts inline)
- Анимации: CSS transitions/keyframes (не тяжёлый JS)
- Производительность важна — бюджет на JS минимален

**Текущий стек:**
- index.html (~1060 строк), чистый HTML/CSS/JS
- Подключается к FastAPI бэкенду через fetch() → Railway
- WebSocket /ws/live для real-time данных
- Компоненты: баланс, позиции, статистика, earn, настройки, логи

---

## Задача

Создай **8 отдельных HTML-файлов** — каждый представляет полноценный визуальный концепт главного экрана Mini App.

Каждый файл должен содержать:
1. Полноценный mock-up главного экрана (статические данные — хардкод)
2. Карточку баланса (USDT + PnL)
3. Строку статуса бота (Autopilot ON/OFF, Q-Score, Fear & Greed)
4. Блок позиций (1-2 фейковые открытые сделки)
5. Нижнюю навигацию (Dashboard / Positions / Earn / Settings)
6. Комментарий в `<style>` с названием концепта и ключевыми цветами

## 8 Визуальных направлений

### 1. `design-cyberpunk.html` — Cyberpunk Neon
**Вайб:** Blade Runner 2049, Ghost in the Shell
- Фон: `#0a0a0f` (почти чёрный с синим)
- Акценты: неоновый пурпурный `#b722ff` + циан `#00fff7`
- Текст: белый с glitch-эффектом на заголовках
- Карточки: тонкая неоновая рамка (box-shadow neon glow)
- Шрифт: monospace (Courier, Space Mono)
- Эффект: scanlines overlay (псевдо-CRT полосы через ::before)
- Числа PnL: пульсирующий glow зелёный/красный

### 2. `design-bloomberg.html` — Bloomberg Terminal Pro
**Вайб:** Bloomberg Terminal, профессиональный трейдинг
- Фон: `#0d0d0d` чёрный
- Акценты: янтарный `#ff9900` + белый `#ffffff`
- Карточки: без рамок, разделители тонкими линиями `#333`
- Шрифт: monospace, плотная типографика, много данных
- Числа: моноширные, выровненные по десятичной точке
- Статус: цветные dots (●) для состояния систем
- Минимум декора — максимум информации

### 3. `design-glassmorphism.html` — Glassmorphism Galaxy
**Вайб:** macOS Big Sur, iOS 15+, космос
- Фон: animated gradient `#0a0020` → `#001040` → `#200020`
- Карточки: backdrop-filter blur(20px) + rgba(255,255,255,0.08)
- Рамки: rgba(255,255,255,0.15) тонкие
- Акценты: electric blue `#4488ff` + soft purple `#9966ff`
- Шрифт: SF Pro / system-ui
- Эффект: subtle noise texture на картах через SVG filter

### 4. `design-synthwave.html` — Synthwave Retro
**Вайб:** OutRun, Miami Vice, 80s aesthetic
- Фон: тёмный `#120024` + горизонт-сетка в перспективе (CSS grid 3D)
- Акценты: магента `#ff006e` + неон-жёлтый `#fbff00` + фиолетовый `#7700ff`
- Заголовки: Audiowide/Orbitron шрифт
- Числа: outline-text (text-stroke) розовый/фиолетовый
- Закаты: linear-gradient в header `#ff006e → #fbff00`
- PnL +: зелёный неон; PnL -: красный неон

### 5. `design-minimal-dark.html` — Minimal Dark (Linear/Vercel style)
**Вайб:** Linear.app, Vercel, Notion Dark
- Фон: `#111111` + карточки `#1a1a1a`
- Акценты: чистый белый + изумрудный `#00d084` для BUY/позитива
- Шрифт: Inter / system-ui, отличная читаемость
- Карточки: border `#2a2a2a`, radius 12px, без теней
- Числа: крупные, жирные, сразу видно PnL
- Анимации: только микро-transitions 150ms ease
- Философия: каждый элемент только если необходим

### 6. `design-matrix.html` — Matrix Terminal
**Вайб:** The Matrix, хакер, Mr. Robot
- Фон: чёрный `#000000`
- Текст: матрица-зелёный `#00ff41` (phosphor green)
- Шрифт: Courier New, monospace строго
- Эффект: матрица-дождь в фоне (лёгкий canvas анимация)
- Карточки: тонкая зелёная рамка + зелёный glow
- Числа: всё в зелёном, отрицательные в `#ff0040`
- ASCII арт в header (логотип QuantumTrade из символов)

### 7. `design-deep-ocean.html` — Deep Ocean Bioluminescence
**Вайб:** глубокий океан, Aurora Borealis, биолюминесценция
- Фон: `#000814` → `#001233` deep navy
- Акценты: биолюминесцентный циан `#48cae4` + мягкий синий `#0077b6`
- Карточки: полупрозрачные, очень тёмные, слабый blue glow
- Числа: светящийся циан для позитива, коралловый `#ff6b6b` для негатива
- Анимации: медленные пульсирующие glow (как медузы)
- Волновой разделитель между секциями (SVG wave)
- Ощущение: спокойно, глубоко, надёжно

### 8. `design-quantum.html` — Quantum Cosmos (Brand Identity)
**Вайб:** QuantumTrade фирменный стиль, научная фантастика
- Фон: deep space `#020010` + звёзды через CSS box-shadow
- Акценты: квантовый фиолетовый `#7c3aed` + золотой `#fbbf24`
- Атом/орбита в логотипе (CSS анимация вращения)
- Q-Score: отображается как "квантовый заряд" с орбитальным индикатором
- Карточки: dark glass `#0d0925` + фиолетовый border-glow
- Числа: золотые для профита, пурпурные для нейтрального
- Философия: это не просто бот — это квантовая торговая система

## Что мне хочется видеть в каждом концепте

1. **Карточка баланса** — крупно, доминирует
   - `$42.30 USDT` (фейк)
   - `+$0.87 (+2.1%)` сегодня

2. **Статус строка** — компактно под балансом
   - `⚛ Автопилот: ВКЛ` | `Q: 82` | `F&G: 12 🔴`

3. **Позиция** — одна открытая сделка
   - `BTC-USDT BUY $5.00` | `+1.8%` | `TP 4% SL 2%`

4. **Пассивный доход** — маленький блок
   - `DCI: 854% APY` | `Earn: 5.2% APR`

5. **Нижняя навигация** — 4 иконки
   - 📊 Дашборд | 💼 Позиции | 💰 Earn | ⚙️ Настройки

## Delivery

- 8 отдельных `.html` файлов в папке `/design-concepts/`
- Каждый — полностью автономный (никаких внешних зависимостей)
- README.md с описанием каждого концепта и рекомендацией по выбору
- Не нужна реальная функциональность — только визуал

**Приоритет выбора**: тот концепт, который выглядит профессионально, читаемо и при этом стильно — это основной UI для ежедневного использования.
